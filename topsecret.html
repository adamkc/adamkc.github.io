Tools & apps

Game · Daily puzzle
Budget Rank

A daily puzzle in the spirit of Wordle: rank five federal policies by how much each would dent the $1.83T federal deficit. Square-by-square feedback on each guess, and a donut showing how much of the deficit your picks would close.

Browser tool
Trail Route Editor

Load a DEM in the browser, drag trail vertices, and run a spring-mass optimizer toward a target grade. Hillshade, contours, 3D terrain, and GeoJSON/KML import/export; everything runs locally.

Interactive map
KMP Meadow Inventory Map

Map + data downloads
Lost Meadows Predictions

A map for browsing and downloading random-forest predictions of lost meadows by HUC10 watershed — high- and medium-confidence polygons and raster outputs across the Sierra Nevada and beyond. The download companion to the machine-learning work on finding lost meadows.

R package + docs
ProcessSpace

Documentation for an R package that processes LiDAR DEMs for stream and meadow restoration — flow paths, process-space polygons, detrended elevation, and KMZ output for Google Earth. Note: the package no longer installs cleanly.
